# ruo
> Yet Another RESTful API framework

## Installation

    npm install --save ruo


## Usage

    const ruo = require('ruo');

    async function main() {
      const app = await ruo.createApplicationAsync()

      app.use(ruo.getRestMiddleware())

      app.listen(8088)
    }

## Development
